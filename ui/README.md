# Keleustes UI (placeholder)

The Keleustes UI is a React + TypeScript frontend that surfaces application
matrix, detail, promotion timeline, release inventory, environment topology,
health, diff, policy blockers, audit, and administration screens (see
PROPOSAL §16).

This directory is a placeholder for the UI workspace. Scaffolding the actual
frontend (Vite + React + TypeScript, design system, API client) happens when
MVP 0 work begins.

Until then, this README exists so the repo layout matches the planned final
shape and so `*.gitignore` rules for `ui/node_modules` and `ui/dist` have a
home.
